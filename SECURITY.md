# Security Policy

## Public Template Scope

This repository is a template. Do not commit:

- Telegram bot tokens
- webhook secrets
- Google Cloud project IDs that expose a private environment
- service account keys
- customer messages or chat identifiers
- deployment logs containing private data

## Reporting

Open a GitHub issue for documentation or template hardening problems. For
private credential exposure, rotate the affected credential first, then report
the issue without posting the secret value.
